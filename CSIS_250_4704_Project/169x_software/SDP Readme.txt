Softwares for SDP-SERIES Switching-Mode Power Supplies (rev.1)

This CD includes 4 softwares.
These are:
1. Control Application Software for interface with your PC.
2. Recalibraion Software for recalibrate the setting Voltages/Currents and LCD displayed Voltages/Currents.
3. Labview Driver
   Note: Before using Labview, please dowload the SDP.llb and save it in C:\Program Files\National Instruments\LabVIEW 6\instr.lib
4. VB Driver

All softwares are compatible with Windows 9x/2K/XP.
Please refer to User Manual for application of these software.
